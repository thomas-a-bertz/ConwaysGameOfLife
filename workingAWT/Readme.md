This version has the following properties:
- it is a working version
- it has a GUI implemented with AWT
- it has a console output
- it has a suite of tests

Compile
-------
- ```mkdir bin```
- ```cd src```
- ```javac -d ../bin/ Board.java```
- ```javac -d ../bin/ Game.java```
- ```cd ..```

Run
---
- ```cd bin/```
- ```java -cp . Game```
- ```cd ..```
